ดาวน์โหลดเอกสาร

รายวิชา การโปรแกรมคอมพิวเตอร์ นี้ คือจุดเริ่มต้นที่สมบูรณ์แบบสำหรับทุกคนที่สนใจในโลกแห่งการเขียนโปรแกรม โดยใช้ภาษา PYTHON ซึ่งเป็นภาษาที่เรียนง่ายแต่ทรงพลัง ผู้เรียนจะได้สัมผัสกับเนื้อหาที่เข้าใจง่าย ครอบคลุมตั้งแต่พื้นฐานจนถึงเทคนิคขั้นสูง พร้อมตัวอย่างโค้ดที่หลากหลาย รวมถึงเนื้อหาใหม่หรือผู้ที่มีประสบการณ์ก็สามารถเรียนรู้และนำไปประยุกต์ใช้ได้จริง

นอกจากเนื้อหาพื้นฐานแล้ว รายวิชายังปูพื้นฐานการประยุกต์ใช้ PYTHON ในด้านต่างๆ เช่น

  • วิศวกรรม: เรียนรู้การจำลองแบบระบบ และการควบคุมอัตโนมัติเบื้องต้น
  • การวิเคราะห์ข้อมูล: ฝึกฝนการสร้างกราฟ และการวิเคราะห์แนวโน้มข้อมูล
  • IoT: เข้าใจหลักการควบคุมอุปกรณ์ และการรับส่งข้อมูลในระบบ IoT
  • AI: การประมวลผลข้อมูลและแสดงผล

ซึ่งจะเป็นประโยชน์อย่างยิ่งสำหรับผู้เริ่มต้นที่ต้องการต่อยอด และนำ PYTHON ไปประยุกต์ใช้ในสาขาต่างๆ ต่อไปในอนาคต ด้วยการเรียนที่เป็นขั้นตอน เข้าใจง่าย และตัวอย่างโค้ด ผู้เรียนจะเข้าใจเนื้อหาได้อย่างรวดเร็ว พร้อมพัฒนาเพื่อการเขียนโปรแกรม PYTHON และก้าวสู่โลกเทคโนโลยีอย่างมั่นใจ

การหาเส้นทางสั้นที่สุด (Travelling Salesman Problem - TSP)
Python

# โปรแกรมหาเส้นทางสั้นที่สุด (Travelling Salesman Problem)
import numpy as np
import itertools

def calculate_distance(point1, point2):
    return np.linalg.norm(np.array(point1) - np.array(point2))

def total_path_distance(path, coordinates):
    distance = 0
    for i in range(len(path) - 1):
        distance += calculate_distance(coordinates[path[i]], coordinates[path[i+1]])
    distance += calculate_distance(coordinates[path[-1]], coordinates[path[0]])
    return distance

coordinates = [
    (0, 0),  # เมือง A
    (1, 5),  # เมือง B
    (5, 2),  # เมือง C
    (6, 6)   # เมือง D
]

paths = itertools.permutations(range(len(coordinates)))
shortest_path = min(paths, key=lambda p: total_path_distance(p, coordinates))
shortest_distance = total_path_distance(shortest_path, coordinates)

print(f"เส้นทางที่สั้นที่สุด: {shortest_path}")
print(f"ระยะทางทั้งหมด: {shortest_distance:.2f}")

คำอธิบายโค้ด:

  • หลักการทำงาน: โค้ดนี้แก้ปัญหา Travelling Salesman Problem (TSP) โดยคำนวณระยะทางที่สั้นที่สุดในการเดินทางผ่านทุกเมืองและกลับมาจุดเริ่มต้น
  • อัลกอริทึม: ใช้การคำนวณระยะทางแบบยูคลิด (Euclidean distance) และทดลองเส้นทางทุกรูปแบบที่เป็นไปได้ (brute force approach)
  • การใช้ไลบรารี: นำ NumPy มาใช้ในการคำนวณระยะทางระหว่างจุด และใช้ itertools สำหรับการสร้างเส้นทางที่เป็นไปได้ทั้งหมด
  • การประยุกต์: แสดงให้เห็นถึงการใช้ Python แก้ปัญหาการหาเส้นทางที่ประหยัดที่สุด ซึ่งใช้ได้กับงานวิศวกรรมหลายด้าน เช่น การขนส่ง โลจิสติกส์ หรือการวางแผนเส้นทาง
ผลลัพธ์การทำงาน:
เส้นทางที่สั้นที่สุด: (0, 2, 3, 1) ระยะทางทั้งหมด: 18.48
การแสดงกราฟฟังก์ชันด้วย Matplotlib
Python
import matplotlib.pyplot as plt
import numpy as np
# ตัวอย่างการแสดงกราฟฟังก์ชัน y = sin(x) x = np.linspace(0, 2 * np.pi, 100)
y = np.sin(x)
plt.figure(figsize=(8, 4))
plt.plot(x, y, label='y = sin(x)', color='blue')
plt.title('Example Graph of Sin(x)')
plt.xlabel('X Axis')
plt.ylabel('Y Axis')
plt.grid(True)
plt.legend()
plt.show()

คำอธิบายโค้ด:

  • หลักการทำงาน: โค้ดนี้สร้างกราฟของฟังก์ชัน sine โดยใช้ไลบรารี Matplotlib และ NumPy ของ Python
  • การสร้างข้อมูล: ใช้ NumPy สร้างข้อมูลจุดต่างๆ ของ x ตั้งแต่ 0 ถึง 2π และคำนวณค่า sin(x) สำหรับแต่ละจุด
  • การใช้ Matplotlib: กำหนดขนาดกราฟ, สร้างเส้นกราฟ, เพิ่มชื่อแกน, ชื่อกราฟ, และเส้นตาราง
  • การประยุกต์: การสร้างภาพกราฟเป็นทักษะสำคัญในงานวิศวกรรมและวิทยาศาสตร์ข้อมูล ช่วยในการวิเคราะห์และนำเสนอข้อมูลได้อย่างมีประสิทธิภาพ
